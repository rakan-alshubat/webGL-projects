CSE 470 HW#2: Pick a Cube

Author: Rakan Alshubat        Date: February 2019

Description:
This program has eight rotating cubes around a central stationary cube. The user can change the rotation speed of the cubes and the size
of the cubes by using the sliders. The user can also click on any of the rotation cubes to make the central cube rotate in the same direction.
The user can reset the central cube back to stationary by clicking the "reset center cube" button

Functionality:
Clicking on an outer cube makes the center cube replicate that rotation.
The center cube can be reset to default of ~no movement~ by clicking the "reset center cube" button.
Using the sliders changes the size and rotation speed of the cubes.

Parameters:
Theta controls the rotation speed of the cubes and is initialized at 0.2
cubeScaler controls the size of the cubes and is initialized to 0.2
